## Lab Overview

This project implements the **Lab 2 Participation Exercise** for CMPUT 301.  
The application extends the *ListyCity* example to allow users to:

- Add new city names to a list
- Delete existing city names from the list
- Interact with the list using basic Android UI components

The focus of this lab is on practicing:
- Object-Oriented Programming (OOP) principles
- Android basics
- `ListView` and `ArrayAdapter`
- Event handling using `OnClickListener`
